Kembali setelah sekian lama meninggalkan wordpress

Leave a Comment / Degree

Masih ingat last time aku main dengan wordpress kalau tak silap tahun 2017 hahaha tu pon bukan betul2 explore just click & drag menu je lepas tu tinggal. So minggu ni merupakan kali pertama aku sentuh wordpress sejak 2017. Totally lupa dah macam mane nak buat tapi jangan bimbang youtube kan ada kah3 .. hail youtube

Jadi untuk kali ni aku guna wordpress latest iaitu 5.3.2. Untuk tema pula aku ambil contoh daripada Astra stater theme. Jadi siapa yang nak try develop website guna wordpress nanti boleh try theme Astra. Serius Astra punya theme banyak yang power dan cantik. Mostly design nampak modern dan tak outdate.

Manakala untuk builder aku guna Elementor. Boleh je kalau nak edit guna default editing tool yang wordpress dah provide, tapi feature dia tak banyak dan bagi aku tak user friendly since aku pon pening nak setup macam mana awal-awal tu. Guna je Elementor kau jadi berkobar-kobar nak try every widget kat dashboard Elementor.

Niat aku buat blog ni hanya untuk mengisi masa lapang sepanjang RMO ni, bayangkan sepanjang tempoh RMO bermula 18hb March hingga 14 April duduk dalam rumah je dan tak keluar. Makan pon makan benda sama. Hari-hari stress main mobile legend sebab banyak team noon dan op. haihh so aku cuba buat blog sebab aku tahu even mudah nak create blog pakai wordpress benda ni still makan masa mungkin berhari-hari sebab proses design ni agak complicated walaupun senang just click, drag & drop je .

Serius aku rasa produktif buat kerja pakai dua display. Yang ni aku guna tv rumah sewa sebagai external dislay untuk display output wordpress yang aku edit

Untuk laptop pula aku pakai laptop mak sedara aku je hahaha processor intek celeron & 4gb ram . takde laptop sendiri lagi doa2kan la aku dapat beli laptop secepat mungkin ..
